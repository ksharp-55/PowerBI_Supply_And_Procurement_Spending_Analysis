# PowerBI_Supply_And_Procurement_Spending_Analysis


## Description
This porject focuses on analyzing procurement spending data using Power BI to identify spending patterns, key suppliers, and cost drivers. Interactive dashboards are created to provide actionable insights for cost optimization and decision-making


## Objective
Delivery a Power BI-driven analysis for procurement spending to improve visibility into supplier costs, spending patterns, and optimization opportunities


## Key Outcomes
-Cleaned and consolidated procurement spend dataset
-Interactive dashboards showing spend by supplier, category, and time
-Identification of top suppliers and high-spend categories
-Documented insights for procurement optimization


## Success Metrics
-Dashboards accurately display and filter procuremnt spend data
-At least 3-5 actionable insights generated from spend patterns
-Stakeholders can quickly identify cost drivery and supplier concentration


## Milestones
1. Complete Data Collection & cleaning
2. Finalize Analysis & Metrics Definition
3. Deliver Power BI Dashboard & Insights
